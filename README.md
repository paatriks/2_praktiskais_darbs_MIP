# Otrais praktiskais darbs kursā "Mākslīgā intelekta pamati".

Darba mērķis ir izpētīt un klasificēt baltvīna kvalitāti (Low/Medium/High), izmantojot nepārraudzītās (hierarhiskā klasterēšana, K-vidējo) un pārraudzītās mašīnmācīšanās metodes Orange rīkā. Datu kopa: [UCI Wine Quality Dataset](https://archive.ics.uci.edu/dataset/186/wine+quality).
